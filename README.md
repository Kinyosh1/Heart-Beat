# Beating Heart Particle System

一个基于 React + Vite + Canvas 开发的极致流畅、数学驱动的跳动爱心粒子系统。

## 特性
- **数学驱动**：使用爱心参数方程生成基础形状。
- **动态同步**：文字缩放、荧光闪烁与心脏跳动频率完美同步。
- **极致流畅**：实时计算粒子位移，支持 60FPS 丝滑动画。
- **色彩美学**：80% 紫色与 20% 粉色粒子交织，搭配红色荧光文字。

## 本地运行

1. **克隆项目**
   ```bash
   git clone <你的仓库地址>
   cd <项目目录>
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **构建生产版本**
   ```bash
   npm run build
   ```

## 技术栈
- React 18
- TypeScript
- Vite (构建工具)
- HTML5 Canvas API
- Tailwind CSS (样式管理)
